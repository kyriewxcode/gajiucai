# gajiucai

#### CIGA2020-48小时极限开发：一款恶趣味的韭菜模拟器

**游戏主题：** 讽刺大数据时代消费陷阱对于普通民众（韭菜）的迫害
**使用引擎：** Unity
**游戏类型：**俯视视角2D弹幕游戏
**游戏风格：**搞怪
**游戏玩法：** 玩家控制角色移动躲避地图中的消费陷阱，同时拾取自己真正想要购买的东西增加心情值尽量延长生存时间。
